My first web page

HTML

Hypertext Markup Language, or HTML, is the basic language of the Interwebs. There are three main languages used in web pages:

  • HTML, which describes the content and layout of a web page,
  • CSS (for Cascading Style Sheet), which describes how the HTML elements appear, and
  • Javascript, which enables us to make dynamic changes to web pages.

HTML is an instance of the Extensible Markup Language (XML). The layout of the page is defined by the HTML elements that the page contains. An HTML element consists of a start tag and an end tag. For example, paragraphs begin with 

<p>

and end with 

</p>
